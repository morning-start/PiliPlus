# PiliPlus 开发文档

## 项目概述

PiliPlus 是一个使用 Flutter 开发的 BiliBili 第三方客户端，支持多平台运行（Android、iOS、Pad、Windows、Linux）。该项目提供了丰富的功能，包括视频播放、直播观看、动态发布、用户交互等，旨在提供比官方客户端更好的使用体验。

## 技术栈

- **框架**: Flutter 3.35.7
- **状态管理**: GetX
- **路由管理**: GetX
- **网络请求**: Dio
- **数据存储**: Hive
- **视频播放**: media_kit
- **弹幕渲染**: canvas_danmaku
- **UI组件**: Material Design

## 项目结构

```
lib/
├── common/           # 公共组件和常量
├── grpc/            # gRPC 相关代码
├── http/            # HTTP API 请求
├── models/          # 数据模型
├── models_new/      # 新版数据模型
├── pages/           # 页面组件
├── plugin/          # 插件相关
├── router/          # 路由配置
├── services/        # 服务层
├── tcp/             # TCP 相关
└── utils/           # 工具类
```

## 核心功能

### 视频播放
- 支持多种画质和音质选择
- 硬件加速
- 弹幕显示和发送
- 字幕支持
- 倍速播放
- 手势控制（亮度、音量、快进快退）
- 记忆播放位置

### 直播功能
- 直播观看
- 弹幕互动
- 礼物、SuperChat 支持
- 直播间表情

### 用户系统
- 多账号支持
- 登录/登出
- 用户信息查看和编辑
- 关注/取关
- 黑名单管理

### 动态系统
- 发布图文动态
- 查看动态列表
- 动态评论和点赞
- 动态转发

### 收藏与稍后再看
- 收藏夹管理
- 稍后再看列表
- 收藏夹排序和整理

### 搜索功能
- 视频搜索
- 用户搜索
- 直播间搜索
- 番剧搜索

## 开发环境设置

1. 确保已安装 Flutter SDK 3.35.7
2. 克隆项目到本地
3. 运行 `flutter pub get` 安装依赖
4. 根据目标平台设置相应的开发环境
